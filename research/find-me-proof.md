# Find Me Proof

Get statistics, data, or evidence to back up a claim or decision.

## Template

```
Find me data to support (or refute) this claim:

"[THE CLAIM OR ASSUMPTION]"

I need:
- 3-5 relevant statistics or data points
- The source for each (company, study, publication)
- How recent the data is
- Any contradicting evidence I should know about

Context:
- I'm using this for [PROPOSAL / PITCH / DECISION / ARTICLE / etc.]
- My audience is [WHO WILL SEE THIS]

Output:
- Bullet points with stat + source + date
- Confidence rating: how solid is this data?
```

## Example

```
Find me data to support (or refute) this claim:

"Most small business owners spend over 10 hours a week on administrative tasks"

I need:
- 3-5 relevant statistics or data points
- The source for each (company, study, publication)
- How recent the data is
- Any contradicting evidence I should know about

Context:
- I'm using this for a pitch to sell my automation services
- My audience is small business owners (1-10 employees)

Output:
- Bullet points with stat + source + date
- Confidence rating: how solid is this data?
```

## When to Use

- Building a business case
- Writing proposals
- Preparing presentations
- Validating assumptions before making decisions
- Content creation that needs credibility
