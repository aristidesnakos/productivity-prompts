# Explain Like I'm Busy

Get a plain-language explanation of something complex without the jargon.

## Template

```
Explain [TOPIC] to me in plain language.

Context:
- I'm a [YOUR ROLE]
- I need to understand this because [WHY]
- My current knowledge level: [BEGINNER / SOME FAMILIARITY / etc.]

Constraints:
- No jargon (or define it if unavoidable)
- Use analogies where helpful
- Keep it under 300 words
- End with "What this means for you:" with 2-3 practical takeaways
```

## Example

```
Explain the new EPA lead paint regulations for renovations to me in plain language.

Context:
- I'm a general contractor
- I need to understand this because I do kitchen and bathroom renovations in older homes
- My current knowledge level: I know lead paint is regulated but not the specifics

Constraints:
- No jargon (or define it if unavoidable)
- Use analogies where helpful
- Keep it under 300 words
- End with "What this means for you:" with 2-3 practical takeaways
```

## When to Use

- New regulations or compliance requirements
- Technical concepts you need to understand quickly
- Explaining something to a client (get the explanation, then forward it)
- Learning about a client's industry before a meeting
