# Project Update

Provide a clear and concise update on project progress.

## Template

```
Write a project update email.

Context:
- Project Name: [PROJECT NAME]
- Current Phase: [PHASE OR MILESTONE]
- Key Updates: [SUMMARIZE IMPORTANT CHANGES, ACHIEVEMENTS, OR OBSTACLES]
- Timeline: [ON TRACK / DELAYED - NEW ESTIMATED COMPLETION]
- Assistance Needed: [ANYTHING SPECIFIC FROM THE AUDIENCE]

Tone:
- Professional and transparent
- Focused and no unnecessary details
- Balanced, highlighting both successes and challenges

Include:
- Brief summary of project status
- Action points or next steps
- Appreciation if applicable
```

## Example

```
Write a project update email.

Context:
- Project Name: Website Redesign Initiative
- Current Phase: Design Review
- Key Updates: Finalizing mockups and incorporating stakeholder feedback
- Timeline: On track for next milestone delivery by February 10th
- Assistance Needed: Feedback from marketing team on mockup #4 by February 5th

Tone:
- Professional and transparent
- Focused and no unnecessary details
- Balanced, highlighting both successes and challenges

Include:
- Brief summary of project status
- Action points or next steps
- Appreciation if applicable
```

## When to Use

- Regular project updates (weekly, bi-weekly, etc.)
- Delivering milestone updates
- Reporting delays or request assistance from stakeholders
